Repository for Unity QuickList Examples

Apr 25th, 2011

This post originally appeared on 2buntu.com, a site that I occasionally write for on varied topics, most of which have to do with my Ubuntu side.

 

One of the cool new features (Among many) in Unity in 11.04 (Hereafter referred to as “Natty”) is QuickLists. QuickLists are this awesome new feature that let you have shortcuts in your Unity Launcher. I was a little bored today, so I created a project on launchpad with some cool examples. So I decided “Hey, lets make a FileZilla quicklist”, so I fired up my trusty vim, and let ‘er rip. So, if you’re interested in getting ahold of these cool new QuickList, just run this command:

sudo apt-get install bzr && bzr branch lp:unity-quicklist-examples

 

 

The first part will install bzr on your system (if you don’t already have it), and the second part will download the files for your perusal. Special thanks go to Octavian Damiean for helping me with the FileZilla QuickList. Any questions/comments/concerns can be sent to james@jamesrgifford.com, or you can file a bug in the Unity QuickList Examples Project.
